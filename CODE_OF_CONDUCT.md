# Code of Conduct

## Our Pledge

We pledge to make participation in this project respectful, welcoming, and harassment-free.

## Expected Behavior

- Use welcoming and inclusive language.
- Respect different viewpoints and experience levels.
- Give and receive constructive feedback.
- Focus on what is best for users and maintainers.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing private information without permission.
- Disruptive or bad-faith participation.

## Enforcement

Project maintainers may remove, edit, or reject contributions that do not align with this code of conduct.
